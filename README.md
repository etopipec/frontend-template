## Frontend Template Project

### Features
- Nuxt 4
- Feature Sliced Design
- ESLint
- Husky
- Primevue

---

### Start
```bash
npm run dev
```

---

### Build
```bash
# .env
npm run build

# .env.prod
npm run build:prod

# .env.test
npm run build:test

```

---

### Linting
```bash
npm run lint

# with fix
npm run lint:fix
```
